i-SHIFT System Fail Safe Action Index
SAE Fault Code (DTC)
Malfunction Criteria
Fail Safe Action
P0504
Diagnosis by the brake information in the F-CAN signal.
  • Received brake signal did not change for 3 drive cycles or more.
    i-SHIFT drive cycle is established under these conditions (Once all of these conditions mentioned are satisfied, 1 drive cycle is established.):
    • -
    Two minutes elapsed after the ignition switch is turned ON.
    -
    Gear is shifted to the 3rd gear or higher.
    -
    Speed has reached 16 km/h or more.
Driving in manual mode is impossible.
P0512
Diagnosis by Starter Switch Signal (STS).
None
P0562
Diagnosis by voltage input.
  • Monitors the VBATT (+B) signal when voltage has reached 9 V or less.
i-SHIFT indicator comes on (When voltage decreases, symptoms such as delay in motor operation occur).
P0563
Diagnosis by voltage input.
  • Monitors the VBATT (+B) signal when voltage reached 28 V or more.
i-SHIFT indicator comes on.
P0603
Diagnosis by the TCM internal circuit check function.
  • Performs ROMSUM check of the Keep Alive Memory (KAM), and assumes a malfunction if the RONSUM value does not match.
  • i-SHIFT indicator comes on.
  • Control is terminated and the condition when a malfunction has been detected (gear position, clutch position) is kept.
P0605
Diagnosis by the TCM internal circuit check function.
  • Checks the RONSUM in the rewritable part of the ROM, and assumes a malfunction if the RONSUM value does not match.
  • i-SHIFT indicator comes on.
  • Control is terminated and the condition when a malfunction has been detected (gear position, clutch position) is kept.
P0666
Diagnosis by the TCM internal temperature sensor signal.
  • When the input value is higher or lower than the designed value.
i-SHIFT indicator comes on.
P0702
Determination by VCC voltage input value.
  • When the input value is higher or lower than the designed value.
  • i-SHIFT indicator comes on.
  • Control is terminated and the condition when a malfunction has been detected (gear position, clutch position) is kept.
P0780
Diagnosis by shift position while shifting gears.
  • Even if synchro position is not reached, but reaching the connecting part in the gear has failed at least three times.
i-SHIFT indicator comes on.
P0810
Diagnosis by the value from the clutch actuator position sensor.
  • When the position sensor value does not change even if the clutch actuator has been commanded to operate.
  • i-SHIFT indicator comes on.
  • When vehicle speed exceedes a certain speed (varies with the speed gear retained), the clutch actuator position when the malfunction occurred is retained, and control is performed.
  • When vehicle speed is below a certain speed, the mode becomes Neutral.
  • Although the engine can be restarted, the time elapsed to start the engine after pressing the starter switch is longer than usual (about 1 sec).
P0882
Diagnosis by input voltage value and F-CAN signal value.
  • When voltage input value in the TCM is almost 0 V, even if the IG1 signal (received F-CAN value) of the ECM is ON.
i-SHIFT indicator comes on (If the F-CAN signal from the ECM is sent continuously, control continues until the IG1 in the ECM turns OFF).
P0883
Diagnosis by input voltage value and F-CAN signal value.
  • When voltage input value in the TCM is almost equal or more than the battery voltage, even if the IG1 signal (received F-CAN value) of the ECM is OFF.
i-SHIFT indicator comes on.
P0904
Diagnosis by sensor signal.
  • Monitors PULSES SELECT signal.
  • When the line for PULSES SELECT signal is open.
  • When the line for PULSES SELECT signal is about 0 V (GND short).
  • When the line for PULSES SELECT signal is almost equal to the battery voltage (BATT short).
  • i-SHIFT indicator comes on.
  • Fixed at the speed gear when a malfunction has been confirmed (connects or disconnects the clutch by vehicle speed).
P0909
Diagnosis by neutral width.
  • When the neutral width of the change actuator is 15.5 mm or above, or 25.5 mm or below
i-SHIFT indicator comes on.
P0910
Diagnosis by sensor signal.
  • Output from the sensor does not change even if the output value in the select actuator position does not detect electrical malfunction in the sensor.
i-SHIFT indicator comes on.
P0914
Diagnosis by sensor signal.
  • Monitors PULSES SHIFT signal.
  • When the line for PULSES SHIFT signal is open.
  • When the line for PULSES SHIFT signal is about 0 V (GND short).
  • When the line for PULSES SHIFT signal is almost equal to the battery voltage (BATT short).
  • i-SHIFT indicator comes on.
  • Fixed at the speed gear at the time a malfunction has been confirmed. Connects or disconnects the clutch by vehicle speed.
P0919
Diagnosis by time spent in neutral position learning.
  • When confirmation of a neutral position is not possible.
i-SHIFT indicator comes on.
P0920
Diagnosis by sensor signal.
  • Output from the sensor does not change even if the output value in the shift actuator position does not detect electrical malfunction in the sensor.
i-SHIFT indicator comes on.
P0942
Diagnosis by in-gear action.
  • When synchro position cannot be achieved even if shift/select is operated and in-gear action is performed of halt condition.
i-SHIFT indicator comes on.
P16C0
CPU performs self-diagnosis while ignition switch is ON, and determines whether the malfunction has been recorded or not.
  • i-SHIFT indicator comes on.
  • Control termination other than alteration of information.
P19E5
Diagnosis by the value from the clutch actuator position sensor.
  • Clutch actuator position sensor value is beyond the range of design value.
  • i-SHIFT indicator comes on.
  • When vehicle speed exceedes a certain speed (varies with the speed gear retained), the clutch actuator position when the malfunction occurred is retained, and control is performed.
  • When vehicle speed is below a certain speed, the mode becomes Neutral.
  • Although the engine can be restarted, the time elapsed to start the engine after pressing the starter switch is longer than usual (about 1 sec).
P19E6
Diagnosis by clutch temperature value.
  • If clutch temperature value reaches 300 °C (572 °F) or more.
Creep termination.
P19E7
Diagnose when Clutch Contact Position 3 Learning was not completed or in progress.
i-SHIFT indicator comes on.
P19E8
Diagnosis using VSPOUT signal.
  • When VSPOUT signal is 0 V or approximately equal to the battery voltage.
i-SHIFT indicator comes on.
P19E9
Diagnosis by ECU BRAKE AND signal.
  • When ECU BRAKE AND signal is 0 V or approximately equal to the battery voltage.
i-SHIFT indicator comes on.
P19EA
Diagnosis by Starter Enable signal return input value.
  • When Starter Enable signal return value is 0 V or approximately equal to the battery voltage.
i-SHIFT indicator comes on.
P19EB

  • Diagnostic software protection not active.
None

P19FA
Diagnosis using F-CAN signal.
  • Torque instability signal received at the TCM is set at 10 minutes. or above.
None
P19FB
Diagnose when Clutch Contact Position 1/2 Learning was completed or in progress.
  • When all phases of learning was not completed.
None
(1st or reverse gear only can be shifted. Drive at very slow speed available.)
P19FC
Diagnosis when Shift Change Actuator Position Learning and Synchronizing Position Learning is completed or in progress.
  • i-SHIFT indicator comes on.
  • Control terminated (Engine startup impossible).
P19FD
Diagnosis by Shift Change Actuator Position Learning value.
  • Shift Position Gear 1/3/5 is +11.4 mm or above, +7.4 mm or below.
  • Shift Position Gear 2/4/6/R is −6.2 mm or above, −10.2 mm or below.
  • Select Position Gear 1/2/ is −2.8 mm or above, −5.8 mm or below.
  • Select Position Gear 3/4/ is +2.1 mm or above, −0.9 mm or below.
  • Select Position Gear 5/6/ is +7.6 mm or above, +4.6 mm or below.
  • Select Position Gear R is +11.95 mm or above, +9.45 mm or below.
  • Synchronizer Position Gear 1/3/5 is +8.2 mm or above, +0.2 mm or below.
  • Synchronizer Position Gear 2/4/6 is +1.1 mm or above, −6.9 mm or below.
A fault code is recorded if one of the conditions above exists.
None
P19FE
Diagnosis by Clutch Contact Position learning value.
  • Clutch Contact Position (long term) is 14 mm or above, or 7 mm or below.
  • Clutch Contact Position (medium term) is 14 mm or above, or 7 mm or below.
  • Clutch Contact Position (short term) is 14 mm or above, or 7 mm or below.
A fault code is recorded if one of the conditions above exists.
None
P19FF
Diagnosis by Friction Coefficient value.
  • Friction Coefficient #1 is 865 or above, or 171 or below.
  • Friction Coefficient #2 is 865 or above, or 171 or below.
  • Friction Coefficient #3 is 865 or above, or 171 or below.
  • Friction Coefficient General is 865 or above, or 171 or below.
A fault code is recorded if one of the conditions above exists.
None
P2785
Diagnosis by motor temperature value and PCB sensor's input value.
  • Motor temperature is 195 °C or above.
  • PCB sensor's input value is 118 °C or above.
A fault code is recorded if one of the conditions above exists.
  • i-SHIFT indicator comes on.
  • Gearshift limit (from 1st to 5th and R only) by Shift map for use during occurrence of malfunction.
P2786
Diagnosis by gear actuator's input energy (calculated value).
  • For a certain period of time, an input of 3000 W or more, or an energy input of 2000 or more occurred.
  • i-SHIFT indicator comes on.
  • Fixed at the speed gear when a malfunction has been confirmed (connects or disconnects the clutch by vehicle speed).
P2787
Diagnosis by clutch temperature value.
  • If clutch temperature value reaches 400 °C or more.
  • i-SHIFT indicator comes on.
  • Creep termination.
U0100
Diagnosis using CAN signal.
  • When TCM cannot receive signals from the ECM.
  • i-SHIFT indicator comes on.
  • When vehicle speed exceedes a certain speed (varies with the speed gear retained), the clutch actuator position when the malfunction occurred is retained, and control is performed.
  • When vehicle speed is below a certain speed, the mode becomes Neutral.
  • Although the engine can be restarted, the time elapsed to start the engine after pressing the starter switch is longer than usual (about 1 sec).
U0103
Diagnosis using CAN signal.
  • When TCM cannot receive signals from gear shift control moule.
  • i-SHIFT indicator comes on.
  • If vehicle speed is over 5 km/h, the position at malfunction is retained, and control is done (change in auto mode from manual mode).
  • When vehicle speed is below 5 km/h, the mode becomes Neutral.
U0155
Diagnosis using CAN signal.
  • When TCM cannot receive signals from gauge control module.
i-SHIFT indicator comes on.
U0401
Diagnosis using CAN signal.
  • When an INVALID value is included in the ECM data received by the TCM.
  • DTC set caused by IAT sensor malfunction/ECT sensor malfunction: None
  • For DTC set caused by other ECM malfunctions: When vehicle speed exceedes a certain speed (varies with the speed gear retained), the clutch actuator position when the malfunction occurred is retained, and control is performed.
  • When vehicle speed is below a certain speed, the mode becomes Neutral.
  • Although the engine can be restarted, the time elapsed to start the engine after pressing the starter switch is longer than usual (about 1 sec).
U0404
Diagnosis using CAN signal.
  • If the data received by the TCM from the gear shift control module contains an abnormal or INVALID value.
Fail safe action is classified according to malfunction mode.
  • Auto/Manual range abnormality: i-SHIFT indicator comes on. However, shifting between Automatic and manual is not possible.
  • For abnormality in other range and abnormality in the data received: i-SHIFT indicator comes on.
  • If vehicle speed is over 5 km/h, the position at malfunction is retained, and control is done. When vehicle speed is below 5 km/h, the mode becomes Neutral.
U1261
Diagnosis using CAN signal.
  • When TCM cannot receive signals from ABS or VSA.
  • i-SHIFT indicator comes on.
  • If the shift lever is not at the N position: Speed gear is fixed at malfunction occurrence. However, to prevent engine stall, clutch opening and closing is controlled.
  • When shift lever is at the N position: Shifts gear to Neutral, and thereafter, Neutral mode is maintained even if shift lever is moved to other positions.
U1400
Diagnosis using CAN signal.
  • Abnormality exists in the ECM data received by TCM.
  • When vehicle speed exceedes a certain speed (varies with the speed gear retained), the clutch actuator position when the malfunction occurred is retained, and control is performed.
  • When vehicle speed is below a certain speed, the mode becomes Neutral.
  • Although the engine can be restarted, the time elapsed to start the engine after pressing the starter switch is longer than usual (about 1 sec).
U1401
Diagnosis using CAN signal.
  • TCM received a shift down request, but when gear is shifted, maximum engine speed is exceeded.
None
U1402
Diagnosis using CAN signal.
  • Before TCM received 5-gear-down request, it received 4-gear-down request.
None
U1403
Diagnosis using CAN signal.
  • INVALID value in the ABS or VSA data received by TCM. When 1 piece of data is included.
  • i-SHIFT indicator comes on.
  • When ABS or VSA unit detects malfunction: Disables downshift when brake is depressed.
  • If ABS or VSA unit did not detect a malfunction: None
U1404
Diagnosis using CAN signal.
  • Abnormality exists in the ABS or VSA data received by TCM.
  • When ABS or VSA unit detects malfunction: Disables downshift when brake is depressed.
  • If ABS or VSA unit has not detect a malfunction: None
U1405
Diagnosis using CAN signal.
  • When multiple INVALID values are included in the ABS or VSA data received by the TCM.
  • i-SHIFT indicator comes on.
  • If shift lever is not at the N position: Speed gear is fixed at malfunction occurrence. However, to prevent engine stall, clutch opening and closing is controlled.
  • When shift lever is at the N position: Shifts gear to Neutral, and thereafter, Neutral mode is maintained even if shift lever is moved to other positions.

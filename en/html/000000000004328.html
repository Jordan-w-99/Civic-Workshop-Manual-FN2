Navigation System Symptom Troubleshooting - Interface dial and/or buttons do not work
Interface dial and/or buttons do not work
NOTE:
  • Confirm the correct version disc is installed in the navigation unit as the wrong DVD or software version can cause a hardware malfunction.
  • Check the vehicle battery condition first.
  • Check the connectors for poor connections or loose terminals.
  • Make sure that the correct version are installed.
  • Inspect the DVD for dirt or damage.
  • Check any official Honda service website for more service information about the navigation system.
  • Before troubleshooting, make sure you have anti-theft codes for the audio system and the navigation system.
  • After troubleshooting, enter the anti-theft codes for the audio system and the navigation system.
  1. Start the vehicle.
  1. Go into the Detail Information & Settings select Unit Check, and use Hard Key test under Unit Check.

  1. Turn the ignition switch to LOCK (0).
  1. Disconnect the interface dial 5P connector.
  1. Check for continuity between interface dial 5P connector terminal No. 3 and body ground.
Is there continuity?
YES -

NO -

  1. Check for continuity between navigation unit connector B (32P) terminal No. 23 and interface dial 5P connector terminal No. 1.
Is there continuity?
YES -
Go to Step 8 .
NO -
There is an open in the circuit between the interface dial and the navigation unit. Replace the affected shielded harness.■
  1. Check for continuity between navigation unit connector B (32P) terminals No. 23 and No. 25.

YES -
There is a short in the wire. Replace the affected shielded harness.

Go to Step 9 .
  1. Substitute a known-good interface dial, and recheck.
Is the system OK?
YES -

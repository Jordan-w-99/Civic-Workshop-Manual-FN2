Camshaft Installation
  1. Install the camshaft (A), then install the camshaft thrust cover (B) with new O-ring (C).
  1. Install the camshaft position (CMP) sensor.
  1. Install the rocker arm assembly.
  1. Install the camshaft sprocket.
  1. Install the harness holder, then install the ground cable.
  1. Install the following engine wire harness connectors and wire harness clamps to the cylinder head:
    • Four injector connector
    • Engine coolant temperature (ECT) sensor 1 connector
    • CMP sensor connector
    • Air fuel ratio (A/F) sensor connector
  1. Install the intake manifold chamber.
  1. Install the air cleaner housing assembly.

M/T Mainshaft Reassembly

Exploded View
Special Tools Required

Driver, 40 mm I.D.
 07746-0030100

Driver, 30 mm I.D.
 07946-MB00000
NOTE: Refer to the Exploded View as needed during this procedure.
  1. Clean all parts in solvent, dry them, and apply lubricant to all contact surfaces except the 3rd/4th and 5th synchro hubs.
  1. Install the new angular ball bearing (A) using the special tool and a press. Note the direction of installation.
  1. Install the needle bearing (A) and 3rd gear (B) on the mainshaft (C).
  1. Install the synchro ring (A) and synchro spring (B).
  1. Install the 3rd/4th synchro hub (A) by aligning the outer synchro ring fingers (B) with the grooves in the 3rd/4th synchro hub (C).
  1. Install the 3rd/4th synchro hub (A) using the special tool.
  1. Install the 3rd/4th synchro sleeve (A) by aligning the stops (B) with the 3rd/4th synchro sleeve and hub. After installing, check the operation of the 3rd/4th synchro hub set.
  1. Install the synchro spring (A).
  1. Install the synchro ring (B) by aligning the synchro ring fingers (C) with the grooves in 3rd/4th synchro hub (D).
  1. Install the 4th gear (A).
  1. Install the needle bearings, distance collar, 5th gear, synchro ring, and synchro spring.
  1. Install the 5th/6th synchro hub (A) by aligning the outer synchro ring fingers (B) with the grooves in the 5th/6th synchro hub (C).
  1. Install the 5th/6th synchro hub (A) using the special tool.
  1. Install the 5th/6th synchro sleeve.
  1. Install the synchro spring and synchro ring.
  1. Install the distance collar, needle bearing, and 6th gear.
  1. Install the new angular ball bearing (A) using the special tool. Note the direction of installation.

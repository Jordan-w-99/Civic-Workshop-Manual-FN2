PTC Heater Core Replacement
Do not apply current when core is removed
  • To prevent electrocution because current will flow through the whole PTC heater core.
  • If any foreign material is stuck, a stench, smoke or fire could occur.

SRS components are located in this area. Review the SRS component locations and the precautions and procedures before doing repairs or service.
  1. Make sure you have anti-theft codes for the audio system.
  1. Make sure the ignition is OFF, then disconnect the negative cable from the battery.
  1. Remove the dashboard.
  1. Disconnect the connector (A), then remove the connector holder (B). Remove the self-tapping screws and PTC heater core (C) from the heater unit.
  1. Install the core in the reverse order of removal, and note these items:
    • Do not pull the PTC heater core harness from a connection to prevent a circuit malfunction or harness burning.
    • Make sure that there is no air leakage.
    • Enter the anti-theft code for the audio system.
    • Reset the power window control unit.
    • Set the clock.

M/T Reverse Lockout Solenoid Disassembly/Reassembly
  1. Write down the frequencies for the radio's preset buttons. Disconnect the negative (−) cable first, then the positive (+) cable from the battery. Remove the battery.
  1. Remove the air cleaner housing.
  1. Remove the battery base.
  1. Disconnect the back-up light switch connector and reverse lockout solenoid connector.
  1. Carefully remove the shift cable, select cable, and cable bracket together so as not to bend the cables.
  1. Remove the bolts (A) and reverse lockout solenoid (B).
  1. Remove the roller (A), the select lock return spring (C), and the select lock cam B.
  1. Install in the reverse order of removal.
  1. Remove the dirt and oil from the sealing surface. Apply liquid gasket (P/N 08C70-K0234M) to the sealing surface.
    NOTE: Do not install components if too much time has passed after applying the liquid gasket (for P/N 08C70-K0234M, no more than 4 minutes, for all others, no more than 5 minutes). Instead, remove the old residue and reapply the liquid gasket.
  1. Install the reverse lockout solenoid.
  1. Install the cable bracket, select cable, and shift cable.
  1. Connect the reverse lockout solenoid connector and the back-up light switch connector.
  1. Install the battery base.
  1. Install the air cleaner housing.
  1. Install the battery. Connect the positive (+) cable first, then the negative (−) cable to the battery.
  1. Enter the anti-theft code for the radio and navigation system, then enter the customer's radio station presets.
  1. Perform the power window control unit reset procedure.

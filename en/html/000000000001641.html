Cylinder Head Installation
  1. Install a new coolant separator in the engine block whenever the engine block is replaced.
  1. Clean the cylinder head and block surface.
  1. Install the new cylinder head gasket (A) and dowel pins (B) on the engine block. Always use a new cylinder head gasket.
  1. Set the crankshaft to top dead center (TDC). Align the TDC mark (A) on the crankshaft sprocket with the pointer (B) on the engine block.
  1. Install the cylinder head on the block.
  1. Measure the diameter of each cylinder head bolt at point A and point B.
  1. If either diameter is less than 10.6 mm (0.42 in.), replace the cylinder head bolt.
  1. Apply engine oil to the threads and under the bolt heads of all cylinder head bolts.
  1. Tighten the cylinder head bolts in sequence to 39 N·m (4.0 kgf·m, 29 lbf·ft). Use a beam-type torque wrench. When using a preset-type torque wrench, be sure to tighten slowly and do not overtighten. If a bolt makes any noise while you are torquing it, loosen the bolt and retighten it from the first step.
  1. After torquing, tighten all cylinder head bolts in two steps (90 ° per step). If you are using a new cylinder head bolt, tighten the bolt an extra 90 °.
    NOTE: Remove the cylinder head bolt if you tightened it beyond the specified angle, and go back to Step 6 of the procedure. Do not loosen it back to the specified angle.
  1. Install the rocker arm assembly.
  1. Install the cam chain.
  1. Connect the engine wire harness connectors, and install the wire harness clamps to the cylinder head.
    • Four fuel injector connectors
    • Engine coolant temperature (ECT) sensor 1 connector
    • Camshaft position (CMP) sensor A (Intake) connector
    • Camshaft position (CMP) sensor B (Exhaust) connector
    • Rocker arm oil control solenoid connector
    • Rocker arm oil pressure switch connector
    • Evaporative emission (EVAP) canister purge valve connector
  1. Install the bolt securing the connecting pipe.
  1. Install the upper radiator hose (A) and heater hoses (B).
  1. Install the harness holder bracket (A), then install the harness holder (B).
  1. Connect the fuel feed hose, then install the quick-connect fitting cover (A).
  1. Install the evaporative emission (EVAP) canister hose (A) and brake booster vacuum hose (B).
  1. Install the exhaust manifold.
  1. Install the intake manifold.
  1. Install the drive belt.
  1. Install the air cleaner housing.
  1. After installation, check that all tubes, hoses and connectors are installed correctly.
  1. Inspect for fuel leaks. Turn the ignition switch ON (II) (do not operate the starter) so that the fuel pump runs for about 2 seconds and pressurizes the fuel line. Repeat this operation two or three times, then check for fuel leakage at any point in the fuel line.
  1. Refill the radiator with engine coolant, and bleed the air from the cooling system with the heater valve open.
  1. Inspect the idle speed.
  1. Inspect the ignition timing.

M/T Assembly Installation
  1. Check that the two dowel pins are installed in the clutch housing.
  1. Apply urea grease UM264 (P/N 41211-PY5-305) to the release fork (A) and the release bearing (B). Install the release fork and the release bearing.
  1. Place the transmission on the transmission jack, and raise it to the engine level.
  1. Install the lower-transmission mounting bolts.
  1. Install the clutch cover.
  1. Install the intermediate shaft.
  1. Install the lower torque rod mounting bolts.
  1. Install the lower torque rod mounting bolts.
  1. Install the driveshafts inboard joint.
  1. Connect the lower arm.
  1. Install the transmission mounting bracket base (A) and stud bolts (B).
  1. Install the transmission mounting bracket bolts (A), transmission mounting bolt (B) and nuts (C) then install the ground cable (D).
  1. Loosen and retighten the lower torque rod mounting bolt.
  1. Install the engine undercover.
  1. Install the upper transmission mounting bolts.
  1. Remove the chain hoist and engine hanger (A) from the engine.
  1. Install the harness holder stay (B) and air cleaner housing bracket (C), then install the breather pipe (D) and the harness holder (E).
  1. Connect the vehicle speed sensor connector (A).
  1. Install the cable bracket (A) and apply a light coat of urea grease UM264 (P/N 41211-PY5-305) to the cable ends (B).
  1. Coat the surface of the wire end rubber with silicon grease then install the cables (C) and the new cotter pins (D).
  1. Connect the back-up light switch connector (A) and the reverse lockout solenoid connector (B), then install the engine harness cover (C) and the harness clips (D).
  1. Apply urea grease UM264 (P/N 41211-PY5-305) to the end of the cylinder rod. Install the slave cylinder. Take care not to bend the clutch line.
  1. Install the air cleaner housing.
  1. Install the battery base.
  1. Install the battery. Connect the positive (+) cable first, then the negative (−) cable to the battery.
  1. Refill the transmission fluid.
  1. Check the clutch operation.
  1. Test-drive the vehicle.
  1. Check the front wheel alignment.
  1. Enter the anti-theft code for the radio and navigation system, then enter the customer's radio station presets.
  1. Perform the power window control unit reset procedure.

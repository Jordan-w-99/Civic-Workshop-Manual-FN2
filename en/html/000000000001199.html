Starter Removal and Installation
Removal
  1. Disconnect the negative cable from the battery first, then disconnect the positive cable.
  1. Remove the front emblem cover.
  1. Disconnect the hood switch connector (A), and remove the harness clamps (B) and water pipe mounting bolts (C).
  1. Remove the radiator upper bracket mounting bolts (A), then remove the bulkhead (B).
  1. Remove the engine cover.
  1. Remove the engine oil pressure switch.
  1. Remove the dipstick.
  1. Remove the dipstick tube (A).
  1. Remove the bolt (B) securing the harness holder, and remove the harness holder clamp (C).
  1. Remove the connector bracket (A), then remove the starter protector (B).
  1. Disconnect the starter cable (A) from the B terminal, then disconnect the connector (B) from the S terminal.
  1. Remove the two bolts securing the starter, then remove the starter.
Installation
  1. Install the starter to the engine. Install the starter cable (A) and connector (B). Make sure the starter cable crimped side of the ring terminal is facing out.
  1. Install the starter protector (A), then install the connector bracket (B).
  1. Install the bolt (A) securing the harness holder, and install the harness holder clamp (B).
  1. Install the dipstick tube (C). Use new O-ring (D).
  1. Install the dipstick.
  1. Install the engine oil pressure switch.
  1. Install the engine cover.
  1. Install the bulkhead (A), then install the radiator upper bracket mounting bolts (B).
  1. Apply body paint to the bulkhead mounting bolts.
  1. Connect the hood switch connector (A), and install the harness clamps (B) and water pipe mounting bolts (C).
  1. Install the front emblem cover.
  1. Connect the positive cable to the battery first, then connect the negative cable.
  1. Start the engine to make sure the starter works properly.
  1. Set the clock.

Engine Installation
  1. Install the accessory brackets and tighten their bolts to the specified torques.
  1. Install the accessory brackets and tighten their bolts to the specified torques.
  1. Raise the vehicle on the lift, and position the engine/transmission under the vehicle. Lower the vehicle, and attach the chain hoist to the engine/transmission, then lift the engine/transmission into position in the vehicle.
    NOTE: Reinstall the mounting bolts and support nuts in the sequence given in the following steps. Failure to follow this sequence may cause excessive noise and vibration, and reduce engine mount life.
  1. Tighten the new side engine mount bracket mounting nuts.
  1. Tighten the new transmission mounting bolt and nuts (A).
  1. Install the ground cable (B).
  1. Make sure the hoist brackets are positioned properly. Raise the lift to full height.
  1. Support the front subframe with the wood piece and a jack, and lift it up to the body.
  1. Tighten the new subframe mounting bolts.
  1. Install the lower torque rod, then tighten the new lower torque rod mounting bolts in the numbered sequence shown.
  1. Make sure the hoist brackets are positioned properly. Lower the vehicle on the lift.
  1. Disconnect the chain hoist from the engine/ transmission assembly.
  1. Remove the hanger.
  1. Raise the vehicle on the lift to full height.
  1. With A/C: Install the A/C compressor.
  1. Install a new set ring on the end of each driveshaft, then install the driveshafts. Make sure each ring ‘‘clicks'' into place in the differential and intermediate shaft.
  1. Connect the suspension lower arms to the knuckles.
  1. Connect the tie-rod end ball joints to the knuckles.
  1. Connect the stabilizer links.
  1. Install the driveshaft heat shield.
  1. Connect the connectors to the steering gearbox.
  1. Install the under-floor three way catalytic converter (under-floor TWC) (A). Use new gaskets (B) and new self-locking nuts (C).
  1. Connect the secondary heated oxygen sensor (secondary HO2S) connector (D).
  1. With A/C: Install the drive belt.
  1. Install the engine undercover.
  1. Lower the vehicle on the lift.
  1. Install the upper radiator hose (A), the lower radiator hose (B), and the heater hoses (C).
  1. Without i-SHIFT: Install the shift cable.
  1. Without i-SHIFT: Install the clutch slave cylinder, and clutch line bracket mounting bolt.
  1. With i-SHIFT: Connect the harness connectors.
  1. With i-SHIFT: Install the harness holder and the harness connector.
  1. With i-SHIFT: Install the clutch slave cylinder.
  1. Align the reference mark (A) on the steering joint and steering gearbox pinion shaft. Connect the steering joint (B) to the steering gearbox pinion shaft (C). Tighten the steering joint bolt.
  1. Install the steering joint cover (D).
  1. Install the steering wheel.
  1. Install the evaporative emission (EVAP) canister hose (A) and brake booster vacuum hose (B).
  1. Connect the fuel feed hose, then install the quick-connect fitting cover (A).
  1. Install the cover (B).
  1. Connect the engine control module (ECM) connectors (A) and engine wire harness connectors (B), then install the harness clamps (C).
  1. Install the ECM (A), then install the ECM cover (B).
  1. Install the air cleaner housing bracket (C).
  1. Install the battery cables (A) to the under-hood fuse/relay box.
  1. Install the harness clamp (B).
  1. Install the air cleaner housing assembly.
  1. Install the under-cowl panel.
  1. Install the air intake duct.
  1. Install the front wheels.
  1. Install the battery. Clean the battery posts and cable terminals with sandpaper, then assemble them, and apply grease to prevent corrosion.
  1. Check that the transmission shifts into gear smoothly.
  1. Inspect for fuel leaks: Turn the ignition switch to ON (II) (do not operate the starter) so the fuel pump runs for about 2 seconds and pressurizes the fuel line. Repeat this operation two or three times, then check for fuel leakage at any point in the fuel line.
  1. Refill the engine with engine oil.
  1. Refill the transmission with fluid:
    • Without i-SHIFT
    • With i-SHIFT
  1. Refill the radiator with engine coolant, and bleed air from the cooling system with the heater valve open.
  1. Do the ECM reset procedure.
  1. Do the crankshaft position (CKP) pattern clear/CKP pattern learn procedure.
  1. Inspect the idle speed.
  1. Inspect the ignition timing.
  1. Check the wheel alignment.
  1. Do the power window control unit reset procedure.
  1. Set the clock.

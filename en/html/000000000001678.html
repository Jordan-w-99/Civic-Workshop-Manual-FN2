Special Tools Required

Sub hanger stay
 07MAK-PY30100

Engine tilt hanger set
 07KAK-SJ40101

Engine hanger stay
 07KAK-SJ40200
These tools use only for except EU models.
  1. Install the accessory brackets and tighten their bolts to the specified torques.
  1. Attach the engine tilt hanger set to the engine as shown. Install the engine into position in the vehicle.
    NOTE: Reinstall the mounting bolts and nuts in the sequence given in the following steps. Failure to follow this sequence may cause excessive noise and vibration, and reduce engine mount life.
  1. Install the side engine mount bracket (A), then tighten the new bolt and nut (B), and tighten the bolt (C).
  1. Install the ground cable (D).
  1. Install the transmission mount (A), then tighten the new transmission mount mounting bolts (B).
  1. Tighten the new transmission mounting bolt and nuts (C).
  1. Disconnect the chain hoist from the engine/transmission assembly, then remove the sub hanger stay.
  1. Raise the vehicle on the lift to full height.
  1. Install the lower torque rod, then tighten the new lower torque rod mounting bolts in the numbered sequence shown.
  1. Lower the vehicle on the lift.
  1. Install the upper torque rod, then tighten the new upper torque rod mounting bolts in the numbered sequence shown.
  1. Raise the vehicle on the lift to full height.
  1. Install the front mount.
  1. Install the front beam with new bolts.
  1. Install the front mount mounting bolt.
  1. Install a new set ring on the end of each driveshaft, then install the driveshafts. Make sure each ring ‘‘clicks'' into place in the differential and intermediate shaft.
  1. Connect the lower arms to the knuckles.
  1. Connect the stabilizer links.
  1. Install the three way catalytic converter (TWC) (A). Use new gaskets (B) and new self-locking nuts (C).
  1. Connect the air fuel ratio (A/F) sensor connector (D) and secondary heated oxygen sensor (secondary HO2S) connector (E).
  1. Lower the vehicle on the lift.
  1. Install the A/C compressor bracket.
  1. Install the A/C compressor (A), then connect the A/C compressor clutch connector (B), and install the harness clamp (C).
  1. Install the heater hoses.
  1. Install the radiator.
  1. Install the idler pulley base.
  1. Install the drive belt.
  1. Install the shift cable.
  1. Install the clutch slave cylinder, and clutch line bracket mounting bolt.
  1. Connect the fuel feed hose, then install the quick-connect fitting cover (A).
  1. Install the evaporative emission (EVAP) canister hose (A) and brake booster vacuum hose (B).
  1. Install the air cleaner housing base bracket (A), then install the air cleaner housing base (B).
  1. Install the harness clamps (C).
  1. Install the engine control module (ECM) bracket, then install the under-hood fuse/relay box to the bracket.
  1. Connect the ECM connectors (A) and engine wire harness connector (B).
  1. Install the harness clamp (C).
  1. Install the ECM (A), then install the ECM cover (B).
  1. Install the battery base (A), then install the harness clamp (B) and ground cable (C).
  1. Install the air intake duct (A), then connect the starter subharness 1P connector (B), and install the harness clamp (C).
  1. Install the battery cables to the under-hood fuse/relay box.
  1. Install the air cleaner housing assembly.
  1. Install the intake air duct (A), then install the vacuum hose (B) and breather pipe (C).
  1. Raise the vehicle on the lift to full height.
  1. Install the engine undercover.
  1. Install the front wheels.
  1. Lower the vehicle on the hoist.
  1. Install the under-cowl panel.
  1. Install the battery. Clean the battery posts and cable terminals with sandpaper, then assemble them and apply grease to prevent corrosion.
  1. Check that the transmission shifts into gear smoothly.
  1. Inspect for fuel leaks: Turn the ignition switch ON (II) (do not operate the starter) so the fuel pump runs for about 2 seconds and pressurizes the fuel line. Repeat this operation two or three times, then check for fuel leakage at any point in the fuel line.
  1. Refill the engine with engine oil.
  1. Refill the transmission with fluid.
  1. Refill the radiator with engine coolant, and bleed air from the cooling system with the heater valve open.
  1. Do the ECM reset procedure.
  1. Do the crankshaft position (CKP) pattern clear/CKP pattern learn procedure.
  1. Inspect the idle speed.
  1. Inspect the ignition timing.
  1. Set the clock.

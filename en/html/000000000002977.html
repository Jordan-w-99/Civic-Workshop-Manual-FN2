CMP Pulse Plate Removal and Installation
Removal
  1. Remove the air cleaner housing assembly.
  1. Remove the intake manifold chamber.
  1. Remove the following engine wire harness connectors and wire harness clamps from the cylinder head:
    • Four injector connector
    • Engine coolant temperature (ECT) sensor 1 connector
    • Camshaft position (CMP) sensor connector
    • Air fuel ratio (A/F) sensor connector
    • Secondary heated oxygen sensor (secondary HO2S) connector
  1. Remove the harness holder mounting bolt and the ground cable, then remove the harness holder from the bracket.
  1. Remove the cylinder head cover.
  1. Remove the camshaft thrust cover.
  1. Hold the camshaft with an open-end wrench, then loosen the bolt.
  1. Remove the camshaft position (CMP) pulse plate.
Installation
  1. Install the CMP pulse plate.
  1. Hold the camshaft with an open-end wrench, then tighten the bolt.
    Specified Torque
    14 x 1.5 mm 34 N·m (3.5 kgf·m, 25 lbf·ft)

  1. Install the camshaft thrust cover (A) with new O-ring (B).
  1. Install the cylinder head cover.
  1. Install the harness holder, then install the ground cable.
  1. Install the following engine wire harness connectors and wire harness clamps from the cylinder head:
    • Four injector connector
    • Engine coolant temperature (ECT) sensor 1 connector
    • Camshaft position (CMP) sensor connector
    • Air fuel ratio (A/F) sensor connector
    • Secondary heated oxygen sensor (secondary HO2S) connector
  1. Install the intake manifold chamber.
  1. Install the air cleaner housing assembly.

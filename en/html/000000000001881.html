M/T Assembly Installation
NOTE: Use fender covers to avoid damaging painted surfaces.
  1. Loosen the upper torque rod mounting bolt (A).
  1. Make sure the two dowel pins (A) are installed in the clutch housing.
  1. Install the release fork, the release bearing, and the boot.
  1. Place the transmission on the transmission jack, and raise it to the engine level.
  1. Install the transmission mounting bolts.
  1. Install the clutch cover.
  1. Install the front engine mount bracket (A) on the transmission and engine.
  1. Install the front beam (A). Loosely install the new front beam mounting bolts (B).
  1. Tighten the front beam mounting bolts in a crisscross pattern in several steps.
  1. Loosely install the new front engine mount bolt (A) and nut (B). Then attach the lower radiator hose to the front engine mount bracket.
  1. Install the intermediate shaft.
  1. Install the driveshafts inboard joint.
  1. Connect the lower arm.
  1. Lower the vehicle.
  1. Install the transmission mount bracket (A).
  1. Install the two upper transmission mounting bolts.
  1. Remove the chain hoist and engine hanger (A) from engine.
  1. Raise the vehicle, and make sure it is securely supported.
  1. Tighten the front engine mounting bolt (A).
  1. Loosen and retighten the lower torque rod mounting bolt.
  1. Refill the transmission with the recommended transmission fluid.
  1. Install the engine undercover.
  1. Lower the vehicle.
  1. Tighten the upper torque rod mounting bolt.
  1. Install the engine control module (ECM) (A).
  1. Install the under-food fuse/relay box (A) on the under-hood fuse/relay bracket (B).
  1. Install the engine harness cover (A).
  1. Connect the back-up light switch connector (A), and the vehicle speed sensor connector (B), the reverse lockout solenoid connector (C) and ground cable (D). Install the harness clips (E).
  1. Apply a light coat of urea grease UM264 (P/N 41211-PY5-305) to the cable ends (A) and coat the surface of the wire end rubber (B) with silicone grease.
  1. Install the cable bracket (C) and the cables (D), then install the new lock pins (E).
  1. Apply urea grease UM264 (P/N 41211-PY5-305) to the end of the cylinder rod. Install the slave cylinder. Take care not to bend the clutch line.
  1. Install the battery base (A) and the harness clip (B).
  1. Install intake air duct (A) and harness clip (B).
  1. Install the air cleaner assembly.
  1. Install the battery. Clean the battery posts and cable terminals. Connect the positive cable to the battery first, then connect the negative cable, and apply multipurpose grease to prevent corrosion.
  1. Install the under-cowl panel and cowl cover.
  1. Test-drive the vehicle.
  1. Check the shift lever and the clutch operation.
  1. Check the front wheel alignment.
  1. Enter the anti-theft code for the radio and navigation system, then enter the customer's radio station presets.
  1. Do the power window control unit reset procedure.

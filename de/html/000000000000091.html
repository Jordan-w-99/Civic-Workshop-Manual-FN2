Audioanlage Fehlercodes
Die Audioanlage zeigt bei einem Problem mit CD-Wechsler, Audio-CD oder dem Diebstahlsicherungscode einen Fehlercode an.
Angezeigter Fehlercode
Mögliche Ursache
Maßnahme
CD DISC
Die CD kann nicht gelesen werden.
CD auswerfen und eine andere CD einlegen.
ERROR
Problem mit dem CD-Player. Häufig sind CD-Labels schuld, die sich im Abspielgerät von der CD abgelöst haben.
Versuchen Sie, die CD auszuwerfen und eine andere CD einzulegen. Wenn das nicht hilft, muss das Audiogerät ausgetauscht werden.
CD HEAT
Der CD-Player ist zu heiß. Das kann passieren, wenn das Fahrzeug den ganzen Tag über in der prallen Sonne steht.
Das Fahrzeug für eine Weile kühler stellen und versuchen, ob der CD-Player wieder geht. Wenn der Fehlercode wieder angezeigt wird, eine andere CD einlegen. Wenn auch das nicht hilft, das Audiogerät austauschen.

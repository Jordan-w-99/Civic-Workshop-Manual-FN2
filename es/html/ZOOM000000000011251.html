Modo de diagnóstico del sistema de navegación
Tabla de mensajes de error
Mensaje de error de la pantalla
Solución
El sistema de navegación no puede conseguir una señal del GPS correcta.
Asegúrese de que no hay nada en el salpicadero que pueda bloquear la antena del GPS. Si no es así, lleve el vehículo a un espacio abierto, lejos de los edificios altos, árboles, etc. Los dispositivos de postventa pueden afectar a la recepción del GPS.
La tapa de la unidad de navegación está abierta o No hay ningún disco DVD introducido. Compruebe, por favor, el sistema.
Asegúrese de que se utiliza el DVD de navegación correcto, con la etiqueta de color turquesa (azul/verde) hacia arriba y la pantalla de navegación cerrada.
Error de lectura del disco DVD (disco DVD incorrecto), consulte a su concesionario.
Verifique el color correcto (turquesa) del DVD instalado.
La temp de la pantalla es demasiado alta. El sistema se cerrará hasta que se enfríe la pantalla.
Cuando la temperatura de la pantalla es demasiado alta, aparecerá brevemente este mensaje y ésta se apagará hasta que baje la temperatura. El sistema se activará de nuevo cuando se haya enfriado la pantalla.
La temperatura exterior es baja, el sistema tardará un rato en activarse.
La temperatura es inferior a -30° C y la ECU del sistema de navegación tiene dificultades para efectuar la lectura del DVD. El sistema se activará cuando suba la temperatura.
Error de lectura del disco DVD (sin formatear), consulte a su concesionario.
Compruebe la existencia de rayas profundas u otros daños en el DVD. Asegúrese de que está utilizando un DVD oficial del sistema de navegación de Honda (color blanco). El sistema no puede leer otras bases de datos de mapas o DVD de vídeo. Si el problema persiste, consulte con su concesionario.
No se ha terminado la ruta. Inténtelo desde una ubicación diferente.
Ruta desde o a un lugar (zona nueva) que no está en la base de datos. Trate de planificar una ruta diferente desde o a un lugar diferente.
No se encuentra una ruta alternativa. Se efectuará la guía de la ruta original.
No se encuentra un método para una ruta alternativa. Debe utilizarse la ruta original.
No se puede encontrar este destino en la base de datos.
No se encontró el destino en la base de datos. Inténtelo con otro destino próximo o seleccione el destino con el joystick.
Una vez puesto en marcha el motor, accione el control del climatizador.
Es necesario que el motor este en marcha para que funcione el sistema de control del climatizador.

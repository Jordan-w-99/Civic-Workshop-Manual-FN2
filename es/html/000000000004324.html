NOTA:
  • Compruebe primero el estado de la batería del vehículo.
  • Compruebe el nivel del volumen de sistema de navegación (consulte el manual del propietario).
  • Revise los conectores para ver si hay terminals sueltos o conexiones defectuosas.
  • Si se ha guardado un código de error de hardware, efectúe primero la localización de averías de ese código.
  • Compruebe que está instalada la versión correcta.
  • Revise el DVD para ver si está sucio o dañado.
  • Antes de iniciar la localización de averías, compruebe que dispone de los códigos antirrobo para el sistema de sonido y para el sistema de navegación.
  • Después de efectuar la localización de averías, introduzca los códigos antirrobo del sistema de sonido y el sistema de navegación.
  • Compruebe que la versión y el color del DVD son correctos.

Códigos de error del sistema de sonido
El sistema de sonido muestra unos códigos de error cuando se detecta un problema con el cambiador de discos, el disco o con el código antirrobo.
Código de error representado
Causa posible
Solución
CD DISC
No se puede leer el disco.
Extraiga el disco e inténtelo con otro.
ERROR
Hay una avería en el reproductor de discos. Un problema bastante común es el despegue de las etiquetas de los discos en el interior del reproductor.
Extraiga el disco e inténtelo con otro. Si persiste la avería, sustituya la unidad de sonido.
CD HEAT
El reproductor de discos está caliente. Esto puede ocurrir si el vehículo se encuentra estacionado todo el día al sol.
Estacione el vehículo en un lugar más fresco durante un cierto tiempo y trate de reproducirlo de nuevo. Si persiste el código de error, inténtelo con otro disco. Si el error sigue persistiendo, sustituya la unidad de sonido.

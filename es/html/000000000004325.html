El control de voz no funciona/no responde
Prueba de diagnóstico: Nivel Mic
NOTA:
  • Compruebe que está instalada la versión correcta.
  • Visite los sitios Web oficiales de Honda para obtener más información sobre el mantenimiento del sistema de navegación.
  • Antes de asumir que una reclamación de voz está relacionada con los componentes, asegúrese de que el sistema de control de voz se está utilizando correctamente.
    • -
    Asegúrese de que está en la pantalla correcta cuando intente emitir un comando de voz. Por ejemplo, el mandato ‘‘Encuentre el restaurante italiano más cercano" solo funciona en la pantalla de los mapas. (En el manual del sistema de navegación encontrará una lista completa de los comandos de voz permitidos para la información que se esté visualizando.)
    -
    Cierre las ventanillas y el techo corredizo.
    -
    Ponga el cambio en 1 ó 2.
    -
    Ajuste el caudal del aire de las salidas de ventilación del aire acondicionado para que no dirijan el flujo del aire contra el micrófono del techo.
    -
    Haga una pausa tras pulsar el botón TALK, y entonces diga un comando de voz con claridad y en un tono natural.
    Si el sistema no reconoce su comando, hable más alto.
    -
    Si el micrófono capta voces diferentes de la suya, el sistemas podría no interpretar correctamente sus comando de voz.
    -
    Si dice un comando con algo en su boca, o si su voz es demasiado ronca, el sistema podría interpretar de forma incorrecta sus palabras.
  • Revise los conectores para ver si hay terminales sueltos o conexiones defectuosas.
  • Compruebe si el micrófono de la consola del techo está suelto; en caso afirmativo, apriételo.
  • Antes de iniciar la localización de averías, compruebe que dispone de los códigos antirrobo para el sistema de sonido y para el sistema de navegación.
  • Después de efectuar la localización de averías, introduzca los códigos antirrobo del sistema de sonido y el sistema de navegación.
  • Determine si el problema le ocurre a una sola persona o a todos los que usan el sistema.
  • Si el sistema tiene problemas sólo con la voz de una persona, se trata de una limitación del sistema.

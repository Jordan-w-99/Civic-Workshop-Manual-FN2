Códigos de Erro do Sistema Áudio
O sistema de áudio mostra códigos de erro quando é detectado um problema com o multi leitor de CDs, o disco de áudio ou o código anti-roubo.
Código de Erro Mostrado
Causa Possível
Solução
CD
Não consegue ler o CD.
Ejectar o CD e tentar com outro.
ERRO
Existe um problema com o leitor de CD. Um problema comum são as etiquetas que se descolam do CD dentro do leitor.
Tente ejectar o CD e tente com outro. Se o problema persistir, substitua a unidade de áudio.
AQUECIMENTO DO CD
O leitor de CD está quente. Isto pode acontecer se o veículo estiver estacionado todo o dia ao sol.
Estacione o veículo num local mais fresco durante um bocado e tente usar o leitor de CD outra vez. Se o código de erro persistir, tente com outro CD. Se o código de erro ainda persistir, substitua a unidade áudio.

Inspecção do Sinal do Sensor APP
NOTA:
  • Este procedimento verifica o sensor APP em posição completamente fechada. Em qualquer outra posição, o sensor APP guarda os DTC descritos nos procedimentos de diagnóstico de avarias.
  • Verifique os DTC Temporários ou DTC com o HDS antes de efectuar o procedimento. Caso sejam indicados DTC, efectue primeiro um diagnóstico de avaria e, só de seguida, o procedimento.
  • Pressione vezes seguidas o pedal do acelerador para verificar o seu funcionamento. Caso não funcione com suavidade, verifique o pedal. No caso de detectar um problema, substitua o módulo do pedal do acelerador.
  1. Ligue o HDS à ficha (A) de análise de dados (DLC) localizada sob o tablier do lado do condutor.
  1. LIGUE a ignição (II).
  1. Certifique-se de que o pedal do acelerador não esteja pressionado, e, de seguida, verifique o SENSOR APP na LISTA DE DADOS com o HDS.
    • Se for 0 ‰, o sensor APP está OK.
    • Se não for 0 ‰, actualize o ECM se este ainda não tiver o software mais recente instalado, ou substitua um ECM em bom estado e, em seguida, passe à Etapa 4 .
  1. Certifique-se de que o pedal do acelerador não esteja pressionado, e, de seguida, verifique o SENSOR APP na LISTA DE DADOS com o HDS.
    • Se for 0 ‰, o sensor APP está OK.
    • Se não for ‰, substitua o módulo do pedal do acelerador , e, de seguida, passe à Etapa 1 .
